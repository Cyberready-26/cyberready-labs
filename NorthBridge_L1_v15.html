NorthBridge Bank — Module 1, Lesson 1
How a Customer Request Travels Through NorthBridge Bank
Select a stage number to follow Rahul’s journey
STAGE 1 Rahul’s Device Where the journey begins

Rahul taps “Check Balance” on his NorthBridge banking app. Before the request travels anywhere, the app authenticates him — fingerprint or PIN. Then it encrypts his request so no one can read it in transit. The sealed, encrypted request is now ready to travel.

STAGE 2 Internet & Web Server The public road and the front door

Rahul’s encrypted request crosses the public Internet — a network no one owns or controls. It arrives at NorthBridge’s Web Server, the bank’s front door. The Web Server receives the request, handles the connection, and routes it inward. It does not store any account data. Its only job is to receive and direct.

STAGE 3 Application Server Authentication and Authorisation

The Web Server passes the request to the Application Server — the brain of the operation. Here two questions are answered. Authentication: who are you? Authorisation: what are you allowed to see? Only after both are confirmed does the Application Server ask the Database for Rahul’s balance.

STAGE 4 The Database The vault

The Database is where NorthBridge stores every account record, transaction, and balance. The Application Server sends a precise query: give me the balance for this account. The Database retrieves exactly that — nothing more. It does not decide what is legitimate. That decision was already made upstream.

STAGE 5 The Return Journey Data travels back to Rahul

The balance travels back through the chain — Database to Application Server to Web Server. The Web Server wraps it in encryption and sends it back across the Internet to Rahul’s phone. The NorthBridge app decrypts it and displays the balance. The entire journey happened in under a second. Rahul saw none of it.

Stage 1 of 5
Key Principle
Before a request leaves the device, it is authenticated and encrypted. Authentication happens on-device. The encrypted request is what travels to the bank.
