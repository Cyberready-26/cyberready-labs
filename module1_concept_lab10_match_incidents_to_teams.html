The Six Teams — Reference Guide
🔴
SOC
Security Operations Centre
Monitors for threats, attacks, and suspicious activity 24×7. Responds to security incidents — unauthorised access, malware, data exfiltration, policy violations.
🔒 Security guards — focused on threats and intrusions
🔵
NOC
Network Operations Centre
Monitors and maintains network infrastructure — routers, switches, firewalls, bandwidth, connectivity. Responds to network outages and performance issues.
🏗️ Road crew — keeps the network pipes flowing
🟢
SRE
Site Reliability Engineering
Owns service availability, performance, and reliability. Responds when applications crash, degrade, or fail to meet their SLA — restoring service as quickly as possible.
🏥 Hospital ICU — responds when systems are critically unwell
🟣
DevOps
Development Operations
Owns the deployment pipeline and application code releases. Responds to deployment failures, build errors, and code-level bugs introduced by a specific release.
🚀 Construction crew — builds and deploys new systems
🟠
IT Operations
IT Operations
Manages physical and virtual infrastructure — server hardware, data centre racks, cabling, power systems, OS patching, and hardware provisioning. Hands-on physical layer work.
🔧 Facilities crew — maintains the physical technology floor
☁️
Platform Eng
Platform Engineering
Provides cloud environment, container platforms, network segmentation, and infrastructure-as-code foundations. Every other team builds on the platform they design and maintain.
🌆 City planners — design the infrastructure everyone else builds on
